Mi Portafolio - Versión lista para GitHub Pages

Archivos incluidos:
- index.html
- style.css
- assets/img/perfil.jpg   (tu foto)
- README.txt

Instrucciones:
1. Descomprime este ZIP.
2. Abre index.html en tu navegador local para revisar.
3. Cambia el email en la sección de contacto si lo deseas.
4. Sube los archivos al repositorio de GitHub (main branch).
   - En GitHub: Add file -> Upload files -> selecciona todos los archivos y commit.
5. Ve a Settings -> Pages y asegúrate de que Source esté en main / (root).
6. Tu sitio estará disponible en: https://<tu-usuario>.github.io/<tu-repo>/
