Archivos incluidos:

- style.css
- img/perfil.jpg

Cómo usar:
1. Descomprime este ZIP.
2. Reemplaza img/perfil.jpg si quieres otra foto (mantén el nombre perfil.jpg).
3. Sube la carpeta al repositorio de GitHub (ej. mi-portafolio).
4. Activa GitHub Pages en Settings -> Pages -> main branch / (root).
5. Tu sitio quedará en: https://<tu-usuario>.github.io/mi-portafolio/

Consejo rápido:
- Edita index.html para actualizar email o enlaces de GitHub/LinkedIn.
